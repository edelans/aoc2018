# how to use

virtual env :

    mkvirtualenv --python=/usr/bin/python3 aoc2018

use this lib ?  https://github.com/wimglenn/advent-of-code-data

    pip install advent-of-code-data

use flake8 for linting

    pip install flake8

and atom linter :

    apm install linter-flake8
